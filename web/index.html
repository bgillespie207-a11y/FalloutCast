Model detail
What do these mean?

The fast planning model is WSEG-10, a documented analytic fallout-planning model (Hanifen 1980): it collapses the atmosphere into one effective wind plus a shear term, so it computes instantly. The layered model advects each altitude slice of the cloud with the live wind at that height, so the plume can curve where winds change with altitude. It needs a live vertical wind profile.

Wind
What is shear?

How much the wind changes per 1,000 ft of altitude (kft = kilofoot). In the fast model, shear is what spreads the fallout pattern crosswind.

The layered model (Tier 1) needs a full vertical wind profile; with a manual wind the API computes the fast model (Tier 0) instead and says so in the notes.

Uncertainty band

Runs the layered model (Tier 1) across live GFS-ensemble wind members and maps the probability that the H+1 dose rate exceeds this level. The outer band is where fallout could reach; the inner is where it very likely will. Uses real ensemble winds (not the model/manual-wind controls); slower than a single plume.

What is an ensemble member?

The GFS ensemble is the global weather forecast run many times with slightly perturbed starting conditions; each run is one "member". More members give smoother probabilities but take longer to compute.

What is fission fraction?

The share of the total yield that comes from fission (the rest is fusion). Fallout radioactivity comes from fission products, so the modeled dose rates scale with this fraction.
