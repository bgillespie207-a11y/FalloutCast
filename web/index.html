Uncertainty band

Runs Tier 1 across live GFS-ensemble wind members and maps the probability that the H+1 dose rate exceeds this level. The outer band is where fallout could reach; the inner is where it very likely will. Uses real ensemble winds (not tier/manual wind); slower than a single plume.

Envelope uses an attack scenario: assumed incoming yields per target class (counterforce ~0.30 Mt, countervalue ~0.50 Mt) — illustrative attacker assumptions, not the target's own weapon.
